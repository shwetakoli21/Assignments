var, let,const 
1. var - var kuthe pn declare karta yeto, function scoped ahe, re-declared karta yeto.
ex., var x = 10; 
     var x = 20;

2.let -  let safe variable ahe , block scoped {} ,cannot be re-declared in same block, update karta yeto.
   "block madhe declare kela ki tya block madhech work karto.Punha declare nahi karta yet"
ex.,  let y= 10;
       y = 20;   //update
      let y = 30;  //redeclare allowed nahii

3.const - update , redeclared and value ekdach assign karta yete 
         const = permanent value. ekda value set kele ki change hot nahi
ex.,  const p = 300;
       p = 200;  //not allowed
