Spring & Spring Boot

What is Spring?

Spring is a powerful, lightweight, open-source Java framework used to build Java applications, especially web applications.

What is Spring Boot?

Spring Boot is a Java framework built on top of Spring that helps create stand-alone, production-ready applications with minimal configuration.

Why Use Spring Boot?

  • Auto Configuration
  • Embedded Servers (Tomcat, Jetty)
  • Starter Dependencies
  • Production Ready Features
  • Fast Development

A simple overview of the most important features of the Spring Framework.

1. Lightweight

Spring has a small footprint and does not require heavy runtime resources. Use only the modules you need.

2. Inversion of Control (IoC)

Spring controls the creation and lifecycle of objects (beans) instead of application code creating them directly.

3. Dependency Injection (DI)

Spring injects required dependencies into objects, reducing coupling and making testing easier.

4. Modular Architecture

Spring is composed of multiple modules (Core, AOP, MVC, Data, Security, etc.). You can pick only the modules you need.

5. Aspect-Oriented Programming (AOP)

AOP lets you separate cross-cutting concerns (like logging, security, transactions) from business logic.

6. Spring MVC

Spring MVC supports building web applications and REST APIs using controllers, request mappings and view resolvers.

7. Easy Integration

Spring integrates smoothly with technologies such as Hibernate, JPA, JDBC, JMS, Kafka and many others.

8. Transaction Management

Declarative transaction management (for example via @Transactional) simplifies database transaction handling.

9. Spring Security

Provides authentication and authorization features (form login, OAuth2, JWT, method security, etc.).

10. Spring Boot Support

Spring Boot makes setup and development fast with auto-configuration, embedded servers (Tomcat/Jetty), and production-ready features.
