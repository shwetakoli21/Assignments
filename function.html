Function Types

What is Anonymous Function?

An anonymous function is a function without a name.

it stored in a variable or passed an argument to other function

it cannot be reused directly unless stored in a variable

Ex:
const display = function () {
    console.log("Anonymous function stored in a variable");
};
display();

What is Arrow Function?

Arrow functions are a shorter and modern way to write functions using =>. They are usually used for simple tasks. 

Ex:
const add = (a, b) => a + b;
console.log(add(2, 3)); 
// Output: 5

What is Callback Function?

A callback is a function passed as an argument to another function.

- It’s a function passed into another function.

- The outer function calls it back at the right time.

- Commonly used in event handling, array methods.

Ex:
function firstTask(name, callback) {
    console.log("Starting first task for " + name);
    callback(); // Run the callback after first task
}

function secondTask() {
    console.log("Second task completed!");
}

firstTask("Shweta", secondTask);

what is Events?

Events are signals that something has happened in the browser. JavaScript can "listen" for these events and run code in response.

Common Event Types:

  • Mouse Events: click, dblclick, mousemove,mouseover, mouseout
  • Keyboard Events: keydown, keyup, keypress
  • Form Events: submit, change, focus,input blur
  • Window Events: load, resize, scroll,unloads
