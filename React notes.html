1. useState in React

Definition: useState is a React Hook that allows functional components to store and update state values.

Syntax:

const [stateVariable, setStateFunction] = useState(initialValue);

Example: Counter

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    

      
Count: {count}

      
    

  );
}
export default Counter;

Notes: Multiple useState hooks can be used. State updates are asynchronous.

2. useEffect in React

Definition: useEffect is a React Hook to run side effects (API calls, timers, event listeners, etc.) in functional components.

Syntax:

useEffect(() => {
  // Your effect code
}, [dependencies]);

Example: Run once on mount

useEffect(() => {
  console.log("Component loaded!");
}, []);

Example: Run when state changes

useEffect(() => {
  console.log("Count changed!");
}, [count]);

Example: Fetch API in useEffect

import { useEffect, useState } from "react";

function UserData() {
  const [users, setUsers] = useState([]);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/users")
      .then(res => res.json())
      .then(data => setUsers(data));
  }, []);

  return (
    

      
User List

      {users.map(u => 
{u.name}
)}
    

  );
}

Cleanup Example:

useEffect(() => {
  const timer = setInterval(() => {
    console.log("Timer running...");
  }, 1000);

  return () => clearInterval(timer);
}, []);

3. Fetch API

Definition: Fetch API is a JavaScript feature to make HTTP requests (GET, POST, PUT, DELETE).

GET Request:

fetch("https://jsonplaceholder.typicode.com/users")
  .then(res => res.json())
  .then(data => console.log(data));

POST Request:

fetch("https://example.com/add-user", {
  method: "POST",
  headers: {"Content-Type": "application/json"},
  body: JSON.stringify({ name: "Shweta", age: 22 })
})
.then(res => res.json())
.then(data => console.log(data));

Modern async/await:

async function getData() {
  const response = await fetch("https://jsonplaceholder.typicode.com/posts");
  const data = await response.json();
  console.log(data);
}
getData();

4. Axios Library

Definition: Axios is a JavaScript library to make HTTP requests. It is promise-based and easier than Fetch API.

Installation: npm install axios

GET Request:

import axios from "axios";

axios.get("https://jsonplaceholder.typicode.com/users")
  .then(response => console.log(response.data))
  .catch(error => console.log(error));

POST Request:

axios.post("https://example.com/api/add", {
  name: "Shweta",
  age: 22
})
.then(res => console.log(res.data))
.catch(err => console.log(err));
